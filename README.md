# 🎵 Spotify Artist Search App

A React-based web app that allows users to search for any artist and view their albums using the Spotify Web API.

---

## 🚀 Features

* 🔍 Search for any artist
* 💿 View artist albums
* 🎨 Clean UI using React Bootstrap
* ⚡ Fast and responsive
* 🔑 Uses Spotify API (Client Credentials Flow)

---

## 🛠️ Built With

* React (Vite)
* React Bootstrap
* Spotify Web API

---

## 📦 Installation

1. Clone the repository:

```bash
git clone https://github.com/your-username/your-repo-name.git
cd your-repo-name
```

2. Install dependencies:

```bash
npm install
```

3. Create a `.env` file in the root directory:

```env
VITE_CLIENT_ID=your_spotify_client_id
VITE_CLIENT_SECRET=your_spotify_client_secret
```

4. Start the development server:

```bash
npm run dev
```

---

## 🔑 Getting Spotify API Credentials

1. Go to the Spotify Developer Dashboard:
   https://developer.spotify.com/dashboard

2. Log in and create a new app

3. Copy your:

   * Client ID
   * Client Secret

4. Add them to your `.env` file
